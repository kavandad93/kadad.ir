== Description ==

Kadad Health Check is a premium-quality WordPress plugin that analyzes the health of your WordPress and WooCommerce websites. It provides a professional dashboard with automatic checks, recommendations, reports, and one-click fixes.

**Key Features:**

* **Comprehensive Health Analysis** - Scans all aspects of your WordPress site including security, performance, SEO, system, WooCommerce, images, database, and updates
* **Visual Dashboard** - Beautiful dashboard with modern cards, progress bars, animated counters, and circular score display
* **One-Click Fixes** - Apply fixes for common issues with a single click
* **Detailed Reports** - Generate beautiful PDF reports, printable reports, and export JSON/CSV data
* **History & Trends** - Save every scan, compare results, and view trends over time
* **Email Notifications** - Get weekly reports and notifications via email
* **Dark Mode** - Modern dark mode support for comfortable viewing
* **WooCommerce Ready** - Full compatibility with WooCommerce stores
* **Translation Ready** - Fully translatable with RTL support
* **Responsive** - Works on all devices

== Installation ==

1. Upload the `kadad-health-check` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the 'Health Check' menu in your WordPress admin
4. Run your first scan and see the results

== Frequently Asked Questions ==

= How often should I scan my site? =

We recommend scanning your site at least weekly to catch issues early. You can set up automatic scanning in the settings.

= Can I use this plugin on a site without WooCommerce? =

Yes! The plugin works on any WordPress site. WooCommerce-specific checks are only performed if WooCommerce is installed.

= Are the one-click fixes safe? =

Yes, all fixes are designed to be safe and reversible. However, we recommend backing up your site before applying major fixes.

= Does this plugin affect site performance? =

No, the plugin is designed to be lightweight and only runs scans when you initiate them or on a scheduled basis.
